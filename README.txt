This repository contains the working demo code from Spring 2026.
This iteration uses AR codes posted in physical space to gauge the user's distance and
movement through space. Below is a breakdown of each file and its contents.

index.html - UI elements and AR code setup

js/ar.js - requests motion and camera permissions, launches AR functionality

js/Dreese.js - implementation of the team's demo path from our demo 3 video documenting a sample
use case of wayfinding in Dreese Labs.

js/nav.js - implementation of an older demo path used for testing and learning purposes. included
as a reference/guide to aid future teams in their implementation.

Here are some other resources and tutorials we found helpful:
https://ar-js-org.github.io/AR.js-Docs/
